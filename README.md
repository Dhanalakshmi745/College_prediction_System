# 🎓 Counselling Prediction System

## 📌 Overview
A full-stack machine learning web application that predicts top college and branch options based on student rank and category.

## 🚀 Features
- Predicts Top 5 college & branch combinations
- Machine Learning model using Scikit-learn
- Flask API for predictions
- Node.js backend integration
- Simple HTML/CSS/JS frontend
- REST API communication

## 🛠️ Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js (Express)
- ML Model: Python, Scikit-learn
- API: Flask
