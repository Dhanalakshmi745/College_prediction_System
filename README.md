# 🎓 Counselling Prediction System

## Problem Statement

Students have a time choosing the right colleges and branches during counselling. This is because they don't get help. Our project is to make a system that uses machine learning to predict the college and branch options for students based on their rank and category.

## 📌 Overview

Building a web application that uses machine learning to predict college and branch options for students based on their rank and category.The results helps them in choice filling process of counselling to get the better college options and to know thw insight of the prediction.Overall,It's a Guidance-driven,not just a raw prediction.

## 🚀 Features

- It predicts the 5 college and branch combinations for students.

- Using a machine learning model built with Scikit-learn.

- The predictions are made through a Flask API.

- Node.js backend for integration.

- The frontend is simple built with HTML, CSS and JavaScript.

- The system uses REST API for communication.

## 🛠️ Tech Stack

- Frontend: HTML, CSS, JavaScript

- Backend: Node.js with Express

- Machine Learning Model: Python with Scikit-learn

- API: Flask

## 📈 Output

The system provides the 5 predicted colleges and branches.
